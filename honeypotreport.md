This report represents the result of live network attacks carried out within an Azure Virtual Machine. <br/>
The primary objective was to deploy Telekom Security's honeypot platform [T-Pot](https://github.security.telekom.com/2024/04/honeypot-tpot-24.04-released.html).<br/>

<br/>
This assessment was carried out using a virtual machine set up in Azure using the Ubuntu Server 20.24 LTS OS. <br/>
